# 辅助组件

## 简述

虽然 Validation API 可直接使用，但无论是在「类组件」或「函数组件」中，通常需要使用内建的几个辅助组件，
来更方便的使用 Mota Validation，在「快速开始」的示例中应该看到过部分使用，这是 Mota Validation 推荐的用法。

目前共有 3 个内建的「辅助验证组件」，包括 `Field`、`Alert`、`State`.
